# INSTALL

```
$ pkg install python
$ pip install bs4
$ pip install requests
$ pip install rich
$ pip install cython
$ rm -rf Brute
$ git clone https://github.com/BOY-H4MZ4H/Brute
$ cd $HOME/Brute
$ python run.py
```
